# 香纳融资服务API文档

## 接口定义

### 融资申请数据接口
1. 发起融资申请
    cc: 新增数据
    
2. 更新融资申请数据
    cc： 更新具体的数据
    
3. 新增融资申请OA审批记录
    cc： 新增OA审批记录
    
4. 融资申请状态更新
    cc： 更新状态为下一状态
         历史状态记录新增状态
         新增确认记录
         
         
5. 融资申请数据查询
    cc: 返回具体的合约数据(合约附件文件数据不返回)
    
6. 融资申请相关文件查询
    cc：返回具体的文件数据
